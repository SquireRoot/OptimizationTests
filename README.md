# OptimizationTests

To run the tests, clone the repo

```
git clone git@github.com:SquireRoot/OptimizationTests.git
cd OptimizationTests/
```

Grab the submodules
```
git submodule init
git submodule update
```

Create a build Directory
```
mkdir build/ && cd build/
```

Build the project
```
cmake ../
make
```

Run the project
```
./OptimizationTests
```
